## 2. 结构改动

### 2.1 移除内容
- 删除 6 张种子卡片
- 新增 1 张说明书卡片（首次启动时自动创建）：
  ```
  类型：qa | 学科：其他
  问题：📖 欢迎使用知识卡片！
  答案：这是一款基于间隔重复的闪卡背诵工具...
  标签：教程
  ```
- 界面提示文字精简（如"点击翻转查看答案" → 移除）

### 2.2 Onboarding 引导页
- 首次启动（localStorage 无 `onboarding-done` 标记时）展示
- 3 屏滑动引导：
  1. "创建知识卡片" + 展示卡片创建界面
  2. "每天背诵复习" + 展示学习模式
  3. "追踪学习进度" + 展示日历和统计
- 底部 Skip / Next 按钮，最后一屏 → "开始使用"
- 完成后设置 `onboarding-done: true`

### 2.3 Markdown 批量导入
- 在"我的"页面设置列表新增「📄 批量导入」
- 支持 Markdown 格式解析：
  ```md
  ## 数学
  Q: 什么是勾股定理？
  A: a²+b²=c²

  Q: [选择] 1+1=?
  A: 2
  - 3
  - 4
  - 5
  ```
- 解析规则：
  - `## 学科名` → 设置后续卡片的学科
  - `Q:` → 问题（可选 `[选择]` 标记为选择题）
  - `A:` → 答案（选择题即为正确选项）
  - `- xxx` → 选择题的错误选项
  - 空行分隔不同卡片
- 导入后显示成功/失败统计
- 同时提供模板下载（生成示例 .md 文件）

### 2.4 界面精简
- 去掉所有冗余的 emoji 装饰文字（如"💡 提示"等大段说明）
- "添加"页面提示文字缩减为一行
- 卡片 Tag 样式改为纯文字 + 小圆点
- "查看全部 →" 改为更小的文字链接
